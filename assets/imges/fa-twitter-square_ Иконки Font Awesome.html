После того, как вы подключили Font Awesome к своему проекту, иконки можно вызывать с помощью тега <i>:

Example of twitter-square fa-twitter-square 
<i class="fa fa-twitter-square" aria-hidden="true"></i>

Примечание: для улучшения доступности сайта, мы рекомендуем использовать aria-hidden="true" для скрытия иконок, используемых только для украшения.
